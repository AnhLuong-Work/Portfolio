Giới thiệu

Lập trình viên Backend với khoảng 2–3 năm kinh nghiệm xây dựng hệ thống microservices .NET chạy production cho nền tảng SaaS đa khách hàng (multi-tenant). Vững C# / ASP.NET Core theo Clean Architecture, CQRS và DDD; thành thạo PostgreSQL, Redis, RabbitMQ/MassTransit và SignalR. Tự làm trọn vẹn nhiều service (chat đa kênh, HRM, quản lý câu lạc bộ, marketing) và tham gia mảng bảo mật lõi nền tảng — IAM, API Gateway và tích hợp thiết bị kiểm soát ra vào. Tư duy thực tế, chú trọng bảo mật, quen làm việc theo PR cùng test và CI/CD.

Service Communication (chat đa kênh) — phụ trách chính
  • Xây dựng hộp thư đa kênh hợp nhất Facebook Messenger, Zalo OA/Cá nhân và widget chat web (hơn 100 API), gửi/nhận tin nhắn real-time qua SignalR.
  • Triển khai chatbot AI dùng pgvector (cosine similarity) + GPT (RAG) để trả lời bám dữ liệu, hạn chế "bịa".
  • Thiết kế chia hội thoại cho agent bằng smooth weighted round-robin; nhận webhook trả 200 OK ngay rồi xử lý bất đồng bộ qua Hangfire; mã hóa credential kênh khi lưu bằng AES-256-GCM.
Bảo mật nền tảng — IAM, API Gateway, Operator
  • Tham gia service IAM (đăng nhập/xác thực, JWT RS256 + JWKS, băm mật khẩu Argon2id, 2FA TOTP, RBAC nhóm/quyền, quản lý session) — hơn 80 endpoint.
  • Làm việc trên API Gateway (YARP): pipeline middleware 12 bước gồm chặn giả mạo header, xác thực JWT / API key, CSRF double-submit, rate limit sliding-window bằng Redis, bơm claims và kiểm tra trạng thái workspace.
  • Đảm bảo cô lập multi-tenant bằng Row-Level Security của PostgreSQL + global query filter của EF Core.
Service HRM & Golf/Quản lý câu lạc bộ — phụ trách
