Giới thiệu

Lập trình viên Backend với khoảng 2–3 năm kinh nghiệm xây dựng hệ thống microservices .NET chạy production cho nền tảng SaaS đa khách hàng (12+ service). Vững C# / ASP.NET Core theo Clean Architecture, CQRS và DDD; thành thạo PostgreSQL, Redis, RabbitMQ/MassTransit, gRPC và SignalR. Đã làm tính năng trải khắp các mảng thương mại (Sales, CRM), fintech (thanh toán online, hóa đơn & hợp đồng điện tử), giao tiếp đa kênh, marketing và bảo mật nền tảng — trong dự án tái nền tảng monolith PHP cũ sang .NET — gồm tích hợp bên thứ ba với VNPay, BKAV, Viettel và Brevo.

Tham gia tái nền tảng monolith PHP cũ thành 12+ microservices .NET (Clean Architecture, shared-kernel dạng NuGet, PostgreSQL, RabbitMQ, gateway YARP, gRPC).
Service Communication (chat đa kênh) — phụ trách chính
  • Xây dựng hộp thư đa kênh hợp nhất Facebook Messenger, Zalo OA, Zalo Cá nhân (bridge ZCA / zca-js bằng Node) và widget chat web (hơn 100 API), real-time qua SignalR.
  • Triển khai chatbot AI (pgvector + GPT RAG); tích hợp chat với CRM và Sales qua gRPC (đồng bộ contact, danh mục sản phẩm).
  • Chia hội thoại bằng smooth weighted round-robin; webhook trả 200 OK ngay rồi xử lý bất đồng bộ qua Hangfire; mã hóa credential kênh bằng AES-256-GCM.
Thương mại — Sales & CRM
  • Xây dựng một phần service Sales (133 endpoint): đơn hàng, hàng hóa/sản phẩm, tồn kho, bảng giá & khuyến mãi, hoa hồng, sổ quỹ và báo cáo.
  • Tham gia service CRM: Lead, Contact, Opportunity, Quotation và analytics (~90 endpoint, expose gRPC).
Fintech — Payment, Hóa đơn & Hợp đồng điện tử
  • Xây dựng service Payment thanh toán online qua VNPay, VNPayQR và VietQR (sinh mã QR, xử lý callback IPN, đối soát).
  • Xây dựng server Hóa đơn điện tử và Hợp đồng điện tử tích hợp nhà cung cấp BKAV và Viettel.
Marketing & Notification
  • Xây dựng engine gửi chiến dịch hàng loạt qua Email/SMS (Brevo) và Zalo ZNS — Brevo API client (xác minh sender, DKIM/SPF), workflow MassTransit, gửi lô Hangfire, render MJML.
  • Xây dựng service Notification (thông báo đa kênh, expose qua gRPC).
Bảo mật nền tảng — IAM & API Gateway
  • Xây dựng API Gateway (YARP) (pipeline 12 bước: chặn giả mạo header, xác thực JWT/API key, CSRF double-submit, rate limit sliding-window bằng Redis) và tham gia IAM (JWT RS256 + JWKS, Argon2id, 2FA TOTP, RBAC — hơn 80 endpoint).
  • Đảm bảo cô lập multi-tenant bằng Row-Level Security của PostgreSQL + global query filter của EF Core.
Gym / HRM, thiết bị & migration hệ thống cũ
  • Phát triển service HRM, Gym và golf/quản lý câu lạc bộ (gói hội viên/subscription, đặt lịch & trừ buổi); tích hợp thiết bị Hikvision (thẻ + Face ID) qua ISAPI + agent Electron giao tiếp SignalR.
  • Chuyển đổi & maintain module Gym và CRM của hệ thống PHP cũ, gồm tích hợp tổng đài / call (FreePBX + Asterisk).
  • Quy trình: làm việc theo PR, viết unit & integration test (xUnit + Testcontainers), chuẩn lỗi RFC 7807, logging có cấu trúc và distributed tracing.
